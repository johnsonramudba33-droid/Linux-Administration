# Day 03 - Linux File Management Commands

## Objective

Learn Linux file management commands used in production environments by
Linux Administrators and PostgreSQL DBAs.

---

# 1. Package Installation

## Purpose

Install software packages from Ubuntu repositories.

### Syntax

```bash
sudo apt install <package_name>
```

### Examples

```bash
sudo apt install ncal
sudo apt install tree
sudo apt install curl
sudo apt install net-tools
```

### Real-Time DBA Usage

- Install monitoring tools
- Install PostgreSQL dependencies
- Install troubleshooting utilities
